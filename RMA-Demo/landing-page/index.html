How It Works

Three simple steps to join the distributed network

1

Download Worker

Install the lightweight worker agent on any machine with Docker. Supports Linux, macOS, and Windows.

2

Connect & Donate

Worker auto-detects your hardware (GPU, CPU, RAM) and registers with the coordinator. Donate compute when idle.

3

Earn Credits

Get priority access to AI services based on your contribution. Track your impact on the dashboard.

Worker Tiers

Automatically assigned based on your hardware

Tier 1

GPU Workers

Requirements: 8GB+ VRAM

  • vLLM inference (2-3x faster)
  • Vision models (OCR, image analysis)
  • Highest priority access
  • Maximum credits earned
Tier 2

Service Workers

Requirements: 4GB+ RAM, 2+ cores

  • RAG services
  • Document processing
  • Entity extraction (NER)
  • Medium priority access
Tier 3

Data Workers

Requirements: 2GB+ RAM

  • Database hosting
  • Cache services (Redis)
  • Vector storage
  • Standard priority access

Quick Start

Start contributing compute in 2 minutes

Terminal 
# Install worker agent
pip install rma-worker

# Start worker (connects automatically)
rma-worker start

# Or specify coordinator
rma-worker start --coordinator COORDINATOR_URL

Alternative: Docker

docker run -d \
  -v /var/run/docker.sock:/var/run/docker.sock \
  -e COORDINATOR_URL=COORDINATOR_URL \
  ghcr.io/rma/worker-agent:latest
Note: Worker agent requires Docker installed. Get Docker →
